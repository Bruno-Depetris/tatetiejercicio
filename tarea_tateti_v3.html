Pedir nombres
string[] jugadores = new string[2];
jugadores[0] = Console.ReadLine();

Turno y bucle principal
El programa repite mientras no haya ganador ni empate. El turno alterna entre 1 y 2 con el operador ternario:
int turno = 1;
while(!hayGanador() && !tableroLleno()) {
// jugada...
turno = turno == 1 ? 2 : 1;
}
Mostrar resultado
Al salir del while, verificar si hubo ganador o empate:
if(hayGanador())
Console.WriteLine("Ganó: " + jugadores[turno-1]);
else
Console.WriteLine("Empate");

Coordenadas
Ingresar jugada
Leer fila y columna. Si la celda vale 0 se puede jugar, si no, avisar y pedir de nuevo:
int fila = int.Parse(Console.ReadLine());
int col = int.Parse(Console.ReadLine());

if(tablero[fila, col] == 0)
tablero[fila, col] = turno;
else {
Console.WriteLine("Posición ocupada");
Console.ReadKey();
}
Dibujar el tablero
Dos for anidados. Convertir número a letra, separar columnas con | y filas con ---:
for(int f = 0; f < 3; f++) {
for(int c = 0; c < 3; c++) {
string v = " ";
if(tablero[f,c]==1) v="X";
if(tablero[f,c]==2) v="O";
Console.Write(" "+v+" ");
if(c < 2) Console.Write("|");
}
Console.WriteLine();
if(f < 2) Console.WriteLine("-----------");
}

Juga y gana
Verificar ganador — filas y columnas
Con un for revisás las 3 filas y las 3 columnas a la vez:
for(int i = 0; i < 3; i++) {
// fila i
if(tablero[i,0]!=0 &&
tablero[i,0]==tablero[i,1] &&
tablero[i,1]==tablero[i,2]) return true;
// columna i
if(tablero[0,i]!=0 &&
tablero[0,i]==tablero[1,i] &&
tablero[1,i]==tablero[2,i]) return true;
}
Verificar ganador — diagonales
Las dos diagonales se chequean fuera del for:
// diagonal ↘
if(tablero[0,0]!=0 &&
tablero[0,0]==tablero[1,1] &&
tablero[1,1]==tablero[2,2]) return true;
// diagonal ↙
if(tablero[0,2]!=0 &&
tablero[0,2]==tablero[1,1] &&
tablero[1,1]==tablero[2,0]) return true;

return false;
